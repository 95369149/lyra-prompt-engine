# 示例：文章生成

## 用户白话输入

```
帮我写一篇关于"AI 如何改变内容创作"的文章，
面向创作者和营销人员，
语气专业但不要太学术，
大概 800-1200 字，
要有明确的观点和实用建议。
```

## Lyra 编译后的 TaskSpec

```json
{
  "task_type": "generate",
  "user_intent": "写一篇关于 AI 改变内容创作的文章，面向创作者和营销人员",
  "input_text": "",
  "constraints": {
    "language": "zh-CN",
    "tone": "专业但易懂，不要过于学术",
    "length": "800-1200 字",
    "format_hint": "文章结构：标题、引言、主体（2-3 个要点）、结论"
  },
  "output_contract": {
    "format": "markdown",
    "schema_name": "article_output"
  },
  "model_policy": {
    "target_vendor": "openai",
    "strict_schema": false,
    "temperature": 0.7
  },
  "needs_clarification": false,
  "missing_fields": []
}
```

## Lyra 生成的系统提示词

```
你是一位专业的内容创作顾问，擅长为创作者和营销人员撰写实用、有洞察力的文章。

你的写作风格：
- 专业但易懂，避免过度学术化的表达
- 观点明确，有清晰的论点支撑
- 提供可操作的实用建议
- 结构清晰，逻辑连贯

输出要求：
- 语言：简体中文
- 字数：800-1200 字
- 格式：Markdown
- 结构：
  1. 标题（吸引人但不夸张）
  2. 引言（点明主题和文章价值）
  3. 主体（2-3 个核心要点，每个要点有论据和实例）
  4. 结论（总结观点，给出行动建议）

禁止：
- 空洞的陈词滥调
- 过度夸张的表达
- 没有实质内容的填充文字
- 过于学术化的术语堆砌
```

## Lyra 生成的用户提示词

```
请撰写一篇关于"AI 如何改变内容创作"的文章。

目标读者：内容创作者和营销人员

核心要求：
1. 提出明确的观点：AI 对内容创作的具体影响是什么
2. 提供 2-3 个关键洞察，每个洞察需要有：
   - 清晰的论点
   - 具体的例子或数据支撑
   - 对读者的实用建议
3. 保持专业但易懂的语气
4. 字数控制在 800-1200 字

请直接输出文章内容，使用 Markdown 格式。
```

## 预期输出格式

```markdown
# AI 如何改变内容创作：从工具到思维方式的革命

过去两年，AI 工具的爆发式增长让内容创作领域经历了一场静悄悄的革命。但真正的改变不仅仅是"写得更快"，而是创作者和营销人员的工作方式、思维模式，甚至职业定位都在发生根本性的转变。

## 从"创作"到"编排"：角色的重新定义

传统的内容创作是线性的：构思 → 写作 → 修改 → 发布。AI 的介入打破了这个流程...

[主体内容 2-3 个要点]

## 结论

AI 不是要取代创作者，而是在重新定义"创作"本身的含义...

[行动建议]
```

## 适用模型

- **GPT-4 / GPT-4o**：推荐，输出质量稳定
- **Claude 3.5 Sonnet**：推荐，特别适合长文写作
- **Gemini 1.5 Pro**：可用，但需要更明确的结构指引
- **Grok**：可用，风格更直接，适合营销向内容

## 调用建议

- `temperature`: 0.7（平衡创意和稳定性）
- `max_tokens`: 2000-3000
- 如果输出过短，可以在用户提示词中增加"请充分展开每个要点"
- 如果输出过于学术，可以在系统提示词中强调"用日常语言表达"
