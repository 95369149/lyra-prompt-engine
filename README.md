# Lyra Prompt Engine

🔴 自进化 AI 提示词编译引擎

Lyra 将模糊的需求转化为结构精确、可直接使用的 AI 提示词。不是静态模板，而是一个会学习、会迭代的编译系统。

## 特性

- 🎯 **6 种任务模式**：文本 / 代码 / 分析 / 创意 / 视觉 / Agent
- 🧬 **三层进化机制**：会话内学习 → 知识刷新 → 版本迭代
- 🎨 **视觉生成协议**：覆盖 Midjourney / DALL-E / Flux / Seedance / Kling / Sora
- 🔍 **模型感知**：自动适配 Claude / GPT / Gemini / 开源模型
- ✅ **质量红线**：8 项可验证的自检清单

## 使用方法

将 `README.md` 中的完整提示词复制到任意 AI 对话的 System Prompt 或首条消息中。

### 快速开始

```
[文本] 帮我写一封拒绝供应商涨价的商务邮件，语气要坚定但不失礼貌
```

```
[视觉] 一只赛博朋克风格的猫在霓虹灯下的东京街头行走
```

```
[Agent] 设计一个客服机器人的系统提示词，处理退款和投诉
```

### 进化指令

| 指令 | 功能 |
|------|------|
| `Lyra /evolve` | 基于会话反馈输出自身优化建议 |
| `Lyra /changelog` | 显示版本变更历史 |
| `Lyra /audit` | 全面自检，报告过时技术 |
| `Lyra /benchmark [模型]` | 针对指定模型输出适配建议 |
